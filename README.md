# 👟 Nike All Star Landing Page

Projeto de **Landing Page de produto** inspirado no modelo **Nike All Star**, desenvolvido com **HTML, CSS e JavaScript**.

A aplicação apresenta um layout moderno para exibição do produto, utilizando imagens, estilos e interações simples para melhorar a experiência do usuário.

---

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

---

## ⚙️ Funcionalidades

* 👟 Exibição do produto Nike All Star
* 🎨 Layout moderno e estilizado
* 🖼️ Imagens do produto
* ⚡ Interações com JavaScript
* 📱 Estrutura simples e responsiva

---

## 📂 Estrutura do projeto

```
nike-all-star
│
├── index.html
├── styles.css
├── scripts.js
│
└── img
    └── (imagens do projeto)
```

---

## ▶️ Como executar o projeto

1. Clone o repositório

```
git clone https://github.com/seu-usuario/nike-all-star.git
```

2. Entre na pasta do projeto

```
cd nike-all-star
```

3. Abra o arquivo

```
index.html
```

no navegador.
